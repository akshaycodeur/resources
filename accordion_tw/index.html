What is the difference between "null" and "undefined" in JavaScript?

In JavaScript, "null" is a value that represents the intentional absence of any object value, while "undefined" means that a variable has been declared but has not been assigned any value. Essentially, "null" is a value that must be explicitly assigned, whereas "undefined" is the default value for uninitialized variables.
